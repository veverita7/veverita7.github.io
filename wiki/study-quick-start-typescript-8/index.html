type system

덕 타이핑

  • 동적 타이핑
  • javascript에서 사용
  • 유연하지만 타입 안전성이 낮음
  • interface를 이용해 안전성을 높일 수 있음

구조 타이핑

  • 컴파일 시간에 타입 검증
  • 구조가 같은 클래스 간에는 서로 타입 호환
  • 구조가 같은 클래스와 인터페이스간에도 호환

구조 서브타이핑

  • 구조가 부분적으로 같아도 타입 호환
  • 하위 타입이 상위 타입으로만 호환
    • 상위 타입: {a: string, b: string}
    • 하위 타입: {a: string, b: string, c: string}
  • 타입없이 구조만 있는 경우에도 동일
  • 함수 타입 간 서브타이핑은 반대로 작동

명목 타이핑

  • java나 c++ 등이 사용하는 타이핑 방식
  • 지정된 타입 간에만 타입 호환
  • 구조 타이핑보다 더 타이트
