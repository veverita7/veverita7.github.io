익명 함수 (152p)

타입스크립트 함수 (156p)

  • javascript와 달리 타입을 설정해 validation 코드를 줄인다.
  • parameter의 기본값을 정할 수 있다. (javasript는 es6 이상)

나머지 매개변수(rest parameter) (160p)

  • 0개 이상의 인수 할당

선택 매개변수(optional parameter) (162p)

  • 0개 이상 1개 이하로 제한된 인수 할당
  • 선택 매개변수와 초기값 설정은 함께 사용 불가

함수 오버로드 (164p)

  • javascript로 변환되어도 동일한 형태를 유지해야해서 오버로드를 독립된 블록으로 선언해서는 안됨
  • 구체적인 인수에서 일반적인 인수 순서로 쌓음
  • 인수 형식에 따라 처리하는 코드 구현

화살표 함수(람다 함수) (169p)

  • python, java 8 이상에서 사용
  • 화살표를 -> 가 아니라 => 사용
  • filter method에 적용
  • reduce method에 적용
  • 자세한 설명 (https://alligator.io/js/finally-understand-reduce)

객체 리터럴 (173p)

  • interface를 이용해 코드 어시스트 지원 가능

익명 함수와 타입 선언 (176p)

  • 익명 함수에 타입 선언 가능
  • 함수와 타입을 분리하여 가독성 상승
  • type 알리아스로 분리하면 재사용 가능

콜백 함수 (178p)

  • 콜백 함수를 호출하는 고차 함수 (higher-order function)
  • 고차 함수 이벤트가 끝난 다음 후속 처리를 하는 콜백 함수 (callback function)
  • 대표적인 예: setTimeout
  • 타입을 외부에서 설정 (콜백 함수가 간결해짐)
  • 콜백 함수를 외부에 설정 (고차 함수가 간결해짐)
