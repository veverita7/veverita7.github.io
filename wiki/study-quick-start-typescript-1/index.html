타입스크립스 스터디 시작

개발은 역시 컴파일을 해야 제맛이라 타입스크립트 공부 시작합니다.
자바스크립트도 컴파일하자!

타입스크립트 특징 (5p)

  • 자바스크립트 + 타입 = 타입스크립트
  • 객체지향 프로그래밍 환경 제공
  • 타입 안정성을 보장

타입스크립트를 공부하는 제일 큰 이유는 타입인거같다.
자바스크립트는 처음 쓸때는 타입이 없어서 편하기는 한데
점점 사이즈가 커지면 불편함이 더 커지는듯하다.
타입스크립트는 타입을 설정해서 코드를 작성하고 컴파일러(tsc)를
통해 자바스크립트 코드로 변환해준다.

이름 ES6 타입스크립트 자바
클래스 class class class
인터페이스 X interface interface
인터페이스구현 X implements implements
상속 extends extends extends
생성자 constructor(){} constructor(){} public 클래스명(){}
